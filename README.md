# Personal Portfolio

A clean, minimal personal portfolio website — ready to deploy on **GitHub Pages** or **Netlify** in under 5 minutes.

Inspired by [Thomas Olsen's Framer template](https://thomasolsen.framer.website/).

---

## 🗂 File Structure

```
portfolio/
├── index.html        ← All sections & content (edit this first)
├── style.css         ← All styles — CSS variables at the top for theming
├── script.js         ← Dark mode, animations, form logic
├── assets/           ← Drop your images here
│   ├── avatar.jpg
│   ├── project-1.jpg
│   ├── project-2.jpg
│   ├── project-3.jpg
│   ├── about-1.jpg
│   ├── about-2.jpg

│   ├── testimonial-2.jpg

│   └── og-image.jpg  ← Social share preview (1200×630px)
├── README.md
└── INSTRUCTIONS.md   ← Step-by-step customization & deployment guide
```

---

## 🚀 Quick Deploy

### GitHub Pages
1. Push this folder to a GitHub repository
2. Go to **Settings → Pages → Source → main branch / root**
3. Your site is live at `https://yourusername.github.io/repo-name`

### Netlify (recommended — free custom domain + form handling)
1. Drag-and-drop the folder to [netlify.com/drop](https://app.netlify.com/drop)
2. Done — you get a live URL instantly
3. Connect to GitHub for auto-deploy on every push

---

## ✏️ Quick Customization

All content is in `index.html`. Every editable spot is marked with a comment:
```html
<!-- EDIT: your name -->
```

For theming, open `style.css` and change the CSS variables at the very top — colors, fonts, spacing — no hunting through the file needed.

See **INSTRUCTIONS.md** for the full step-by-step guide.

- Fully responsive (mobile, tablet, desktop)
